# HV Distribution Design

## Overview

This repository contains the conceptual design of an 11 kV electrical distribution network for a multidisciplinary scientific research campus.

The project is being completed as an engineering portfolio project to demonstrate power systems design, electrical engineering calculations, technical documentation, AutoCAD drawings, Python programming and professional version control using Git.

---

## Project Objectives

- Design an 11 kV campus distribution network
- Calculate maximum demand
- Select network topology
- Size transformers
- Size 11 kV cables
- Perform voltage drop calculations
- Perform fault level calculations
- Develop protection philosophy
- Develop earthing philosophy
- Assess reliability and future expansion
